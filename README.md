# 🌍 Geo Tool – Geocoding, Distance Calculator & Interactive Map

A modern, user‑friendly desktop application built with **PyQt6** and **Folium** that provides:

- **Address → Coordinates** – Geocode any address to latitude/longitude using OpenStreetMap (Nominatim).
- **Coordinates → Address** – Reverse geocode to obtain a human‑readable address.
- **Distance between addresses** – Calculates the shortest (geodesic) distance between two places.
- **Distance between coordinate pairs** – Direct distance between two lat/lon points.
- **Interactive map** – Automatically displays both points and a route line on an interactive Leaflet map (generated with Folium).

## ✨ Features

- 🧠 **Multi‑threaded** – Never freezes the interface while querying the geocoding service.
- 🗺️ **Live map** – After a distance calculation, the map tab updates with markers and the connecting line.
- 🎨 **Modern UI** – Clean CSS styling, rounded corners, hover effects, and intuitive layout.
- 🔁 **Repeatable workflows** – Each tool can be used multiple times without leaving the tab.
- 📏 **Metric & imperial** – Distances shown in both kilometers and miles.
- ⚠️ **Error handling** – Validates inputs, handles network issues, and provides clear messages.
